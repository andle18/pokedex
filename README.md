Pokémon Game Boy Explorer
Project overview
A small static frontend demo that turns the Pokémon API into a handheld-style search experience. It is designed to demonstrate UI interaction, DOM updates, and external API consumption using vanilla HTML, CSS, and JavaScript.

Features
Search Pokémon by name or number
Navigate Pokémon sequentially with Game Boy-style controls
Toggle between Pokémon types and first moves
Display the current Pokémon’s evolution chain
Themed Game Boy-style visual presentation
Tech stack
HTML, CSS, JavaScript
Client-side API consumption with fetch
Static hosting friendly, no build step required
Architecture
index.html contains the markup and UI structure
style.css contains all presentation and theming
index.js handles state, API requests, event handling, and DOM updates
Installation & setup
Clone the repository
Run:
npm run dev
Open:
://localhost:5500
