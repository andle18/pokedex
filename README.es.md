Explorador Game Boy de Pokémon
Resumen del proyecto
Demo frontend estática que transforma la API de Pokémon en una experiencia al estilo Game Boy. Está pensada para mostrar capacidad de diseño de interfaz y consumo de APIs con HTML, CSS y JavaScript puros.

Funcionalidades
Buscar Pokémon por nombre o número
Navegar entre Pokémon con controles estilo Game Boy
Alternar entre tipos y primeros movimientos
Mostrar cadena de evolución del Pokémon actual
Diseño visual tematizado de consola portátil
Stack tecnológico
HTML, CSS y JavaScript puro
Consumo de API con fetch
Frontend estático sin paso de compilación
Arquitectura
index.html define la estructura y el UI
style.css define el estilo y el layout
index.js gestiona estado, peticiones a la API, eventos y renderizado
Instalación y configuración
Clona el repositorio
Ejecuta:
npm run dev
Abre:
://localhost:5500

No se requieren variables de entorno.

Scripts disponibles
